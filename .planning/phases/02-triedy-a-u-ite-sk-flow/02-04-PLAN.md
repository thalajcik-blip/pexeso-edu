**Create `src/components/teacher/ClassResults.tsx`:**

Props: `classId: string`, `assignment: AssignmentWithDeck`

**Data fetching:**
1. Get class member user_ids from `classroomStore.members`
2. Query `game_history` for those users filtered by the assignment:
   - If `assignment.set_slug`: `supabase.from('game_history').select('user_id, quiz_correct, quiz_total, duration_sec, played_at').in('user_id', memberIds).eq('set_slug', assignment.set_slug).order('played_at', { ascending: false })`
   - If `assignment.custom_deck_id`: same but `.eq('custom_deck_id', assignment.custom_deck_id)`
3. Group results by `user_id`. For each student, take their best attempt (highest quiz_correct/quiz_total ratio).
4. Join with member profiles to get username and avatar_id.

**Display:**
- Table with columns: Student (username + avatar), Score (quiz_correct/quiz_total as percentage), Duration (duration_sec formatted as Xm Ys), Played At (date formatted cs-CZ)
- Class average row at the top (bold): average of all students' best scores
- Students with no results show "-" in all columns

**Color coding (DASH-04):**
- Green (`#22c55e`): score >= 70%
- Yellow/amber (`#eab308`): score 40-69%
- Red (`#ef4444`): score < 40%
- Apply as background-color at 15% opacity on the score cell, text color solid

**CSV export (DASH-05):**
- Button "Exportovat CSV" at top right
- `downloadCSV` function: builds array of objects `{ Student, Score_Percent, Quiz_Correct, Quiz_Total, Duration_Sec, Played_At }`
- Uses `Blob` with `text/csv;charset=utf-8;` MIME type
- Creates download via `URL.createObjectURL` + `<a>` click trick
- Filename: `{className}_{deckTitle}_{YYYY-MM-DD}.csv`

**Localization:** CS/SK/EN labels for column headers, "no results", export button text.
