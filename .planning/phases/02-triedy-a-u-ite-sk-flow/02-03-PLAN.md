**1. Create `src/components/student/JoinClassRoute.tsx`:**

This component handles the `/join/:code` route. The flow:

a) Extract invite code from URL path (`/join/PX-XXXX` -> `PX-XXXX`). Use `window.location.pathname.split('/join/')[1]`.

b) Look up the class: `supabase.from('classes').select('id, name').eq('invite_code', code).maybeSingle()`. If not found, show error "Trida nebyla nalezena" with a link back to `/`.

c) Check auth state via `useAuthStore`:
   - If `user` is null: store invite code in `sessionStorage.setItem('pexedu_pending_join', code)`, then call `useAuthStore.getState().openAuthModal('login')`. Show message "Pro pripojeni k tride se prihlaste".
   - If `user` exists: proceed to join.

d) Join the class: `supabase.from('class_members').upsert({ class_id: classData.id, user_id: user.id }, { onConflict: 'class_id,user_id' })`. The UPSERT prevents duplicate rows on re-join.

e) On success: show success message "Pripojeni k tride [class name]!" with a sonner toast, then redirect to `/` after 2 seconds using `window.location.href = '/'`.

f) Add a `useEffect` that watches `user` — when user transitions from null to a value (post-login), check `sessionStorage.getItem('pexedu_pending_join')`. If found, look up class by code and run the UPSERT. Clear sessionStorage after.

g) Localize messages for CS/SK/EN using `useGameStore(s => s.language)`.

**3. Also add pending-join check in `App.tsx`** — After the main app loads, check if `sessionStorage.getItem('pexedu_pending_join')` exists and `user` is set. If so, execute the join flow (same UPSERT logic) and clear the sessionStorage item. This handles the case where user logs in from the auth modal and the page re-renders as the main App rather than the JoinClassRoute.

Add a `useEffect` in App.tsx watching `user` from authStore:
```typescript
useEffect(() => {
  const pendingCode = sessionStorage.getItem('pexedu_pending_join')
  if (!pendingCode || !user) return
  (async () => {
    const { data: cls } = await supabase.from('classes')
      .select('id, name').eq('invite_code', pendingCode).maybeSingle()
    if (!cls) { sessionStorage.removeItem('pexedu_pending_join'); return }
    await supabase.from('class_members').upsert(
      { class_id: cls.id, user_id: user.id },
      { onConflict: 'class_id,user_id' }
    )
    sessionStorage.removeItem('pexedu_pending_join')
    toast.success(`Pripojeni k tride ${cls.name}!`)
  })()
}, [user])
```
