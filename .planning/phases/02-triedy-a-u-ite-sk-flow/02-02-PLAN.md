**1. Create `src/store/classroomStore.ts`** — Zustand store with:

State fields:
- `classes: ClassWithStudentCount[]`
- `currentClassId: string | null`
- `members: ClassMemberWithProfile[]`
- `assignments: AssignmentWithDeck[]`
- `loading: boolean`

Actions:
- `fetchClasses()` — query `classes` where `teacher_id = user.id`, join count of `class_members` and max `last_active_at`. Use Supabase `.select('*, class_members(count), class_members(last_active_at)')` or two separate queries. Store result in `classes`.
- `createClass(name: string)` — generate invite code using `crypto.getRandomValues()` with charset `ABCDEFGHJKLMNPQRSTUVWXYZ23456789` (no 0/O, 1/I), format `PX-XXXX`. INSERT into `classes` with `teacher_id: user.id`, `name`, `invite_code`, `gdpr_confirmed_at: new Date().toISOString()`. On conflict (23505 error on invite_code), retry with new code (max 3 retries). After success, call `fetchClasses()`.
- `fetchClassDetail(classId: string)` — set `currentClassId`, query `class_members` joined with `profiles` (username, avatar_id) for this class. Query `class_assignments` for this class. Store in `members` and `assignments`.
- `assignDeck(classId: string, setSlug: string | null, customDeckId: string | null)` — INSERT into `class_assignments`. After success, refetch assignments.
- `removeAssignment(assignmentId: string)` — DELETE from `class_assignments`. Refetch.

Import `supabase` from `../services/supabase`, types from `../types/classroom`, and `useAuthStore` for `user.id`.
